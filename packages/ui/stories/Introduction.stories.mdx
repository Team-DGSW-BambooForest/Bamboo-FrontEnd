# 대대숲 디자인 system

대나무숲 프론트엔드 개발팀의 대부분의 디자인은 이 system을 사용합니다.

## 설치

yarn으로 설치하기

```sh
yarn add @bamboo/ui
```

npm으로 설치하기

```sh
npm i @bamboo/ui
```

## 사용하기

사용하자 하는 ui 컴포넌트를 import하여 사용

```tsx
import { Button } from "@bamboo/ui";

const App = () => {
  return (
    <Button size="md" color="black">
      button
    </Button>
  );
};
```
